# **Open Non-Commercial License (ONC)**  

![License Badge](https://img.shields.io/badge/License-ONC%201.0-blue)  
*A permissive open-source license that allows free use, modification, and distribution—but prohibits commercial use.*  

---

## **📜 What is ONC?**  
The **Open Non-Commercial License (ONC)** is a free and open-source software (FOSS) license that:  
✅ **Allows** unrestricted copying, modification, and distribution.  
✅ **Requires** retention of copyright notices and license files.  
🚫 **Prohibits** commercial use (unless explicit permission is granted).  

It is designed for developers who want to **share their work freely** while preventing **unauthorized commercialization**.

## **❓ FAQ**  

### **1. Can ONC be used for non-software works (e.g., docs, art)?**  
Yes! ONC applies to **code, documentation, media, and derivatives**, as long as the terms are followed.  

### **2. What counts as "commercial use"?**  
Any use intended for **direct or indirect financial gain**, including:  
- Selling the software or derivatives.  
- Including it in a paid product/service.  
- Using it for business/internal profit-driven purposes.  

### **3. How is ONC different from MIT/CC BY-NC?**  
| Feature       | MIT       | CC BY-NC  | ONC       |  
|--------------|----------|-----------|-----------|  
| **Allows commercial use?** | ✅ Yes | ❌ No | ❌ No |  
| **Requires attribution?** | ✅ Yes | ✅ Yes | ✅ Yes |  
| **Designed for code?** | ✅ Yes | ❌ No | ✅ Yes |  

ONC is **like MIT but with a non-commercial clause**, making it ideal for **open-source code that should stay non-commercial**.  

### **4. Can I dual-license ONC with another license (e.g., GPL)?**  
Yes, but **ONC’s non-commercial clause will still apply** unless the user complies with the alternative license.
